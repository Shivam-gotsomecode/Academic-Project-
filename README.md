# AcademiaCRM

A CRM dashboard designed for B2B Academia sales management.

## Features
- Lead Management
- Sales Pipeline Tracking
- Follow-up Automation
- AI Intelligence Dashboard
- Analytics and Reporting

## Technology Used
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run
1. Download the repository.
2. Open `AcademiaCRM.html` in any modern web browser.
3. Start using the dashboard.
